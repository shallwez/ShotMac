# QuickMarkShot

一个简单的原生 macOS 截图标注工具。

## 使用

1. 双击 `build/QuickMarkShot.app` 启动；应用会在后台常驻，并在菜单栏显示“截”。
2. 首次运行时，在系统设置中允许屏幕录制权限。
3. 按 `Option+A`，拖动鼠标选择截图区域。
4. 截图会立即进入剪切板，也可以继续添加文字、矩形、箭头或画笔标注。
5. 在截图上双击，合成标注、更新剪切板并关闭窗口。

应用运行后常驻菜单栏，可从“截”图标重新截图或退出。

## 构建

```sh
make app
```
